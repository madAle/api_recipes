<p align="center">  
  <img width="50" height="60" src="api_recipes.png"/>
</p>

Consume HTTP APIs with style
   
## Installation

Add this line to your application's Gemfile:

```ruby
gem 'api_recipes'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install api_recipes
    
## Compatibility

Ruby 1.9.3 or higher.
